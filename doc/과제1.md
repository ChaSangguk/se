# 대상 시스템 품질 요소 추정하기

- 클라우드 파일 공유 시스템

1. 사용자가 원하는 기대치는 무엇인가?

- 사용자는 파일을 업로드, 다운로드 할 수 있고, 공유할 수 있어야 한다.

- 파일을 업로드 할 시 파일의 이름, 수정날짜등의 정보가 표시되어야 한다.

- 특별한 프로그램 다운로드 없이 웹 브라우저 만으로 접근 할 수 있어야 한다.

- 파일별로 읽기, 수정 등 권한을 세부화하여 부여할 수 있고, 외부와 보안을 유지하며 링크로 공유할 수 있어야한다.

- 파일을 이름으로 빠른 시간내 검색할 수 있어야 하며, 특정 기간 이후, 특정 사용자가 업로드한 파일, 특정 파일 형태등으로도 검색할 수 있어야한다.

2. 개발자 혹은 유지보수 담당자가 원하는 기대치는 무엇인가?
3. 개발 조직의 관리자가 원하는 기대치는 무엇인가?

- 관리자는 사용자를 그룹으로 묶어 관리할 수 있어야 한다.

- 계정을 생성, 수정, 삭제 할 수 있어야 한다.

- 로그인시 인증기능, 파일 전송시 암호화, 권한 기반 접근 제어등의 방법으로 보안을 보장할 수 있어야한다.
